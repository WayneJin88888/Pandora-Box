# Pandora-Box 基本使用

## 1、点击配置导入订阅
- 支持 clash、v2ray、sing-box订阅解析
- 支持 vless://... vmess://... hysteria2://... trojan://... 等分享链接，可多行，直接输入解析
- 支持 yaml、base64、json 直接输入解析
